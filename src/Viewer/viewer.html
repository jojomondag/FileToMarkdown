Welcome to FileToMarkdown Viewer

A simple way to view and edit your markdown files

Getting Started

  1. Drop markdown files onto the sidebar dropzone
  2. Click the sidebar dropzone to browse for files
  3. Select a file from the sidebar to view its content

Features

  • View markdown files with syntax highlighting
  • Edit markdown content directly
  • Organize files in a folder structure
  • Quick file navigation
